# Emeka — Spark Builders

A personal 3rd-grade learning plan for **Emeka** (age 8), built the **Spark Builders** way:
*a kid facing outward — noticing real problems and building real fixes.*

Built from her real 2nd-grade report card: huge strengths in **effort, persistence, speaking, and the
arts**, with core academics (reading comprehension, writing, math facts) **currently below grade level**
— so the plan is **steady foundational reps that feel like building**, aimed at exactly what her
teacher named, to climb from a **2 → 3**.

## The idea in one line
Two engines run every week: **daily skill reps** on her app
([emeka-grades.vercel.app](https://emeka-grades.vercel.app)) + **build expeditions** where she makes
something real for a real person.

## The two superpowers she learns
- 🔭 **Foresight** — *see it coming.* Spot the smoke before the fire and build a small early stop.
- 🛠️ **Respond** — *fix what's here.* Find the real gap under a problem and build the fix.

She runs the Spark loop 4 times this year — two as Foresight, two as Respond (Foresight first,
Respond as the capstone).
